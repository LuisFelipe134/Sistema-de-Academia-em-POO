# 🏋️ Sistema de Academia

Projeto desenvolvido em Python para praticar os conceitos de Programação Orientada a Objetos (POO).

## 📚 Conceitos utilizados

- Classes e Objetos
- Herança
- Polimorfismo
- Classes Abstratas (`ABC` e `@abstractmethod`)
- Encapsulamento
- Composição
- Métodos especiais (`__str__`)
- Organização de código

## ⚙️ Funcionalidades

- Adicionar treinos
- Listar treinos cadastrados
- Executar todos os treinos
- Cálculo estimado de calorias gastas
- Menu interativo no terminal utilizando **Rich**

## 🛠️ Tecnologias

- Python 3
- Rich

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git
```

2. Instale a biblioteca necessária:

```bash
pip install rich
```

3. Execute o projeto:

```bash
python main.py
```

## 📌 Objetivo

Este projeto foi criado como forma de estudo para consolidar os conceitos de Programação Orientada a Objetos em Python através de um sistema simples de gerenciamento de treinos.
